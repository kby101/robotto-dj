# robotto-dj
Just another Discord.js music bot.

Features include:
 - queue system
 - play youtube link
 - search on youtube
 - shuffle
 - skip
 - pause/resume
 
WIP features:
 - save playlists
 - load playlists
 
 Depends on the following Node modules:
 - discord.js
 - fs
 - ytdl-core
 - youtube-node
